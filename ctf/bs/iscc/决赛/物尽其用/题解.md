# 物尽其用 (Make the Most of Things)

**ISCC 2022 决赛 · Misc**

## 题目描述

> 附件是一个 ZIP 文件 `TrainModel.zip`，内含 `Input.png`、`model_state.npz`、`config.json` 和 `train.exe`。需要从这些文件中找到 Flag。

---

## 题目附件分析

### 1. 解压附件

```
TrainModel.zip
├── Input.png          # 512×512 RGB 图像，LSB 隐写了数据
├── model_state.npz    # NumPy 模型状态文件，包含 hint 和模型参数
├── config.json        # 训练配置（无关紧要）
└── train.exe          # 训练程序（无需逆向）
```

### 2. 核心数据：`model_state.npz`

该文件包含了以下关键字段：

| 字段 | 用途 |
|------|------|
| `base_hint_bytes` | 56 字节，base64 + gzip 编码的 hint |
| `base_aux_meta` | `b"b64+gz"`，指示编码方式 |
| `base_conv1_weight/bias` | CNN 第一层卷积参数（无用） |
| `base_fc1_weight/bias` | 全连接层参数（无用） |
| `final_mean/final_std` | 数据归一化参数（无用） |
| `obs_raw` / `obs_trans` | 观测数据（无用） |

解码 hint：

```python
import gzip, base64

encoded = state["base_hint_bytes"]
hint = gzip.decompress(base64.b64decode(encoded)).decode()
# => "ISCC{MD5(flag1R||flag2G)}"
```

**Hint 含义**：最终 flag 为 `ISCC{MD5(flag1 || flag2)}`，其中 `flag1` 来自 R 通道，`flag2` 来自 G 通道（经过映射）。

### 3. 隐写载体：`Input.png`

512×512 的 RGB 图像，每个颜色通道的 **LSB（最低有效位）** 隐藏了 `ISCC{...}` 格式的 payload（最多 16 字节）。

---

## 解题思路

### Step 1 — LSB 提取

从 R、G、B 三个通道分别提取 LSB：

```python
def extract_lsb_payload(img, channel, max_bytes=16):
    arr = np.array(img.convert("RGB"))
    bits = (arr[:, :, channel] & 1).reshape(-1)
    bits = bits[:max_bytes * 8]
    return np.packbits(bits, bitorder="big").tobytes()
```

提取结果：

| 通道 | Payload |
|------|---------|
| R | `ISCC{ZmoKmIaCZp}` |
| G | `ISCC{BZunamPQaL}` |
| B | `ISCC{jIYrHdVhWr}` |

### Step 2 — 通道映射分析

Hint 揭示了通道映射关系：`(R, G, B) → (G, B, R)`

即逻辑上的输入位置与物理通道之间被故意打乱：
- **逻辑 R 位置** → 物理 **G** 通道
- **逻辑 G 位置** → 物理 **B** 通道
- **逻辑 B 位置** → 物理 **R** 通道

结合 hint `flag1R || flag2G`，可知：
- `flag1` 来自逻辑 R = **物理 R 通道**（恰好一致）
- `flag2` 来自逻辑 G = **物理 B 通道**（需要映射）

### Step 3 — B 通道数据解扰

B 通道的 payload `ISCC{jIYrHdVhWr}` 不是直接可用的，其中间部分被伪随机重排打乱了。

打乱使用固定种子 `2239801264` 的 `numpy.random.RandomState.shuffle()`：

```python
CHANNEL_SEEDS = {
    "R": 863148490,
    "G": 72199925,
    "B": 2239801264,
}

def scramble_between_markers(data, seed):
    """对 ISCC{...} 中间部分进行伪随机重排"""
    start = data.find(b"ISCC{")
    end = data.find(b"}", start + 5)
    middle = data[start+5:end]
    
    indices = np.arange(len(middle), dtype=np.int64)
    np.random.RandomState(seed).shuffle(indices)
    shuffled = middle[indices]
    
    return data[:start+5] + shuffled.tobytes() + data[end:]
```

对 B 通道 payload 应用相同 seed 的解扰：

```
输入:  ISCC{jIYrHdVhWr}
种子:  2239801264
排列:  [5, 2, 8, 7, 3, 9, 4, 1, 6, 0]
输出:  ISCC{dYWhrrHIVj}
```

得到 `flag2 = dYWhrrHIVj`。

### Step 4 — 提取 flag 中间部分

从 R 通道的 payload 中提取 flag 中间部分：

```python
def extract_flag_inner(data):
    text = data.decode()
    start = text.index("{") + 1
    end = text.rindex("}")
    return text[start:end]
```

- `flag1 = ZmoKmIaCZp`（来自 `ISCC{ZmoKmIaCZp}`）
- `flag2 = dYWhrrHIVj`（来自解扰后的 `ISCC{dYWhrrHIVj}`）

### Step 5 — 计算最终 Flag

根据 hint 的指示，最终 Flag 是 `flag1` 与 `flag2` 拼接后的 MD5：

```python
digest = hashlib.md5((flag1 + flag2).encode()).hexdigest()
final_flag = f"ISCC{{{digest}}}"
```

```
flag1 = ZmoKmIaCZp
flag2 = dYWhrrHIVj
concat = ZmoKmIaCZpdYWhrrHIVj
MD5(ZmoKmIaCZpdYWhrrHIVj) = abcacd4276967a3089ca27506882ad2d
```

## EXP 代码

```python
#!/usr/bin/env python3
import base64, gzip, hashlib, zipfile
from io import BytesIO
import numpy as np
from PIL import Image

CHANNEL_SEEDS = {"R": 863148490, "G": 72199925, "B": 2239801264}
CHANNEL_INDEX = {"R": 0, "G": 1, "B": 2}

def extract_lsb_payload(img, channel, max_bytes=16):
    arr = np.array(img.convert("RGB"))
    bits = (arr[:, :, channel] & 1).reshape(-1)[:max_bytes * 8]
    return np.packbits(bits, bitorder="big").tobytes()

def scramble_between_markers(data, seed):
    start = data.find(b"ISCC{")
    end = data.find(b"}", start + 5)
    if start < 0 or end < 0:
        return data
    left = data[:start + 5]
    middle = np.frombuffer(data[start+5:end], dtype=np.uint8).copy()
    right = data[end:]
    indices = np.arange(len(middle), dtype=np.int64)
    np.random.RandomState(seed).shuffle(indices)
    return left + middle[indices].tobytes() + right

def solve(zip_path):
    with zipfile.ZipFile(zip_path) as zf:
        img = Image.open(BytesIO(zf.read("Input.png")))
        state = np.load(BytesIO(zf.read("model_state.npz")))

    hint = gzip.decompress(base64.b64decode(bytes(state["base_hint_bytes"]))).decode()
    print(f"[*] Hint: {hint}")

    payloads = {ch: extract_lsb_payload(img, CHANNEL_INDEX[ch]) for ch in "RGB"}
    for ch, p in payloads.items():
        print(f"[*] Input {ch}: {p.decode()}")

    # Channel mapping: (R,G,B) -> (G,B,R)
    # flag1 from R, flag2 from B (after descramble)
    flag2_payload = scramble_between_markers(payloads["B"], seed=CHANNEL_SEEDS["B"])
    print(f"[*] Descrambled B → flag2: {flag2_payload.decode()}")

    flag1 = payloads["R"].decode().split("{")[1].split("}")[0]
    flag2 = flag2_payload.decode().split("{")[1].split("}")[0]

    digest = hashlib.md5((flag1 + flag2).encode()).hexdigest()
    return f"ISCC{{{digest}}}"

print(solve("TrainModel.zip"))
# => ISCC{abcacd4276967a3089ca27506882ad2d}
```

## 总结

| 考点 | 说明 |
|------|------|
| **LSB 隐写** | 图像每个像素各通道最低位嵌入数据，肉眼不可见 |
| **Hint 提取** | 从 NPZ 文件中提取 base64+gzip 编码的提示信息 |
| **通道映射** | 物理 RGB 通道与逻辑位置被故意错位，需要根据 hint 推理 |
| **伪随机重排** | 固定 seed 的 `RandomState.shuffle()` 可逆（两次相同排列恢复） |
| **MD5 签名** | 两个 flag 片段拼接后取 MD5 作为最终 flag |

该题的核心技巧在于 **利用题目自身提供的模型文件来获得解题提示**，而非直接逆向训练程序，体现了「物尽其用」的命题意图——把附件中每个文件的价值都挖掘出来。
